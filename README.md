# jest-playwright-examples
## What is Playwright?
The playwright is a Node.js library to automate Chromium, Firefox, and WebKit with a single API. Playwright is built to enable cross-browser web testing.

**Steps:**
>1. npm install
>2. npm run test:jest
>3. npm run test:playwright
